# 변수 type
+ void
  + 존재하지 않음(없음) 
+ boolean
  + true or false
  + 논리형 타입 
+ char
  + 문자 한개(정수 포함)
+ unsigned char
  + 문자 한개(부호 없는 숫자 포함)
+ byte
  + 1byte 크기의 숫자 
+ int
  + 2byte 크기의 숫자
+ unsigned int
  + 2byte 크기의 부호 없는 숫자
+ word
  + unsigned int와 동일
+ long
  + 4byte 크기의 숫자
+ unsigned long
  + 4byte 크기의 부호 없는 숫자
+ short
  + int와 동일
+ float
  + 4byte 크기의 실수
+ double
  + 4byte 크기의 실수
+ string
  + char 형 배열
+ String
  + 문자열
+ array
  + 모든 타입으로 구성할 수 있는 배열



## 지역변수와 전역변수
+ 차이
  + 변수를 블록 내에서만 사용 가능하느냐 아니면 전체에서 사용 가능하느냐의 차이
    <br>
    -> scope
+ 전역변수는 가장 바깥쪽에 변수가 선언되어 있어 파일 내 어느곳에서 사용할 수 있는 변수
+ 지역변수는 어떤 블록안에서 변수가 선언되어 해당 블록 안에서만 사용가능한 변수
