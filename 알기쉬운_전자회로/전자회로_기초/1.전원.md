# 전원이란?
+ 전자회로에서 전기적인 에너지를 공급하는 장치
+ 전압원
  + 공급하는 전기적인 에너지를 전압을 기준으로 해서 공급하는 전원을 말한다.
+ 전류원 
  + 주변에 접속된 회로에 관계없이 일정한 전류를 회로에 공급하는 전원을 말한다.

## 교류 전압원과 직류 전압원
+ 가정에서 콘센트를 가전에 사용했을때 전압이 변하지 않는다. 이와같이 공급하는 전류의 크가와 관계없이 일정한 전압을 전압원이라고 하고, 이 상황에서는 가정에 공급하고 있는 전압이 교류이기 때문에 이것을 교류 전압원이라고 한다. 이러한 상황이 아닌 직류로 전압을 공급하면 이것은 직류 전압원이 된다.
+ 교류 전압원은 AC 어댑터(어? 그럼 DC 어뎁터도 있나?)라는 기기를 사용해 직류 전압원으로 바꿀수 있다.

## 주파수와 주기
+ 정현파는 $$v = V_msin wt$$로 나타낸다.
+ 1주기 
  + 파형이 양, 음의 변화를 완전히 하여 처음 상태로 되돌아 갈때까지의 변화를 1주기라고 한다.
+ 주기와 주파수 사이의 관계
  + f = $\frac{1}{T}$[HZ]
+ 교류에서는 1주기의 길이에 관계 없이 1주기의 각을 항상 360˚로 정하고 있으며 이와 같이 표시되는 각을 전기각이라고 한다.
## 최대치와 실효치

  + 교류와 같은 일을 하는 직류 전압의 크기로 그 교류전압의 값을 나타내는 것이 가능하다.

## 위상

## 전압 · 전류의 표시법
+ 아래 그림에서는 전압 · 전류의 표시법을 나타냈다. 2개의 점 $$a, b$$ 사이의 전압을 나타내는 경우 $$b$$로 부터 $$a$$에 화살표를 하고 전압치 $$V$$ 혹은 $$v$$를 화살표의 중간에 써 넣는다.
  +  $$b$$로 부터측정한 $$a$$의 전압이 $$V$$ 혹은 $$v$$ 임을 나타냄
  +  직류전압에서 $$V$$의 값이 정(+)인 경우 $$a$$가 $$b$$ 보다 전압이 높은 것을 의미한다.
    + 만일 낮은 경우 $$V$$의 값을 부(-)로 하던가 혹은 화살표의 방향을 거꾸로 한다.
    + 교류에서는 전압의 정 · 부가 시간과 함께 변화하지만 어느 시각을 기준으로 해서 전압의 방향을 화살표로 나타낸다.
      <br>
![image](https://github.com/user-attachments/assets/53481676-2d81-4c0d-8e7f-1ec3082b42c4)

+ 위 그림에서 전류를 나타내는 경우는 $$a$$로부터 $$c$$로 전류가 흐르고 있을때 전류가 흐르고 있는 선을 따라서 화살표를 $$a$$로부터 $$c$$로 쓴다.
  + 교류전류도 전압과 같은 방법으로 어느시각을 기준으로해서 전류가 흐르고 있는 방향을 화살표의 방향으로 한다.
  + 전류가 흐르는 방향이 다를 경우 전류값을 부(-)의 값으로 한다.
+ $$V,I$$는 직류(교류에서 실효치) $$v,i$$는 교류(순시치)


+ 두개의 직류 전압원 $$V_1,V_2$$를 아래 그림과 같이 접속하는 걸 직렬 접속이라한다.
  + $$a-b$$간의 전압 $$V$$는 $$V = V_1 + V_2$$ 로 되고, 합성 전압은 2개의 전압원의 합이다.
+ 낮은 전압의 전압원을 다수 사용하여 높은 전압의 전압원을 만들 때 사용하는 접속법을 병렬 접속이라함
  + $$V_1,V_2$$의 값이 다르면 각각의 전압원이 다르기 때문에 $$a-b$$ 간의 전압은 $$V_1,V_2$$의 어느 쪽의 전압이 되어도 모순이 된다.
    + 전압원들의 병렬접속은 불가능하다.

## 주석
#### 정가 시간, 부가 시간
+ 정가 시간: 처음 시작 시삭
+ 부가 시간: 시작 이후에 지나간 시간
### 정현파
+ 수식 
  + $$x(t)=Acos(2πf_0t+ϕ)=Acos(ω_0t+ϕ)$$
+ 좌표평면 위에서 주기적인 모양을 갖는 *개곡선
+ 사인파, 코사인파가 포함
+ 본질운 원 위의 회전에 관한 것이다. 그리고, 시간에 따른 회전을 기술한 것이다.
+ 원 위의 회전에 맞춰 x 축 y 축의 시간 변화를 각각 그래프로 표현하면 정현파를 얻을 수 있다.
+ 정현파의 3가지 특성 값
  + 진폭(amplitude)
    + 정현파의 진폭은 회전하는 원의 반지름의 길이와 같다.
  + 주파수(frequency)
    + 주파수의 개념은 회전 속도와 관련되어 있다. 얼마나 빨리 회전하는지에 관한 것이다. 빨리 회전할 수록 주파수는 높다.
    + 라디안 각도 체계
      + Degree 는 1회전=360단위
      + Radian 은 1회전=2π
      + Gradian 은 1회전=400단위
    + 라디안 주파수(ω)
      + ω = 2πf
  + 위상 변이(phase shift)
    + cos파로 표현했을때는 위/아래 sin파로 표현했을때는 좌우로 그래프가 움직이게 되는 현상
    + 간단히 말해 "원의 어디서부터 출발할 것인가" 이다.
