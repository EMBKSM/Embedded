# I2C 통신이란?
+ Inter-Integrated Circut(I2C)는 IC 사이 통신 링크를 제공하는 양방향 2와이어 직렬 버스 이다
  + 두가닥으로 직렬 통신을 하는 통신프로토콜
+ 구성
  + 하나의 마스터와, 다른 하나의 슬레이브로 구성
  + 슬레이브는 127개까지 구성이 가능
    + 메인 *ECU가 있으면, 그 외 여러가지 디바이스들이 묶여 통신이 가능
